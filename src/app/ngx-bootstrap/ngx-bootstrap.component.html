ngx-ui-tour is a UI tour library built for Angular. It's inspired by angular-ui-tour.

TourNgxBootstrapModule is an implementation of the tour UI that uses NgxBootstrap popovers to display tour steps.

Installation
  1. npm install ngx-ui-tour-ngx-bootstrap bootstrap ngx-bootstrap
  2. Import TourNgxBootstrapModule into your app module If you are using v9 or older, you have to import TourNgxBootstrapModule.forRoot()
  3. Make sure RouterModule is imported in your app module
  4. Add bootstrap SCSS to your styles.scss:
  5. Make sure Ngx Bootstrap is installed correctly
